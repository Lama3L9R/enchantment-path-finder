### 一款用于查找最少铁砧使用次数的附魔方案查找

---

请先安装必须环境 `nodejs` 确保指令 `npm` 位于环境变量中，然后执行以下指令安装 `yarn`:
```
$ npm -g install yarn
```

构建方法 **(确保你有 `nodejs` `yarn` 环境)**:
```
$ yarn build
```

开发用实时预览:
```
$ yarn start
```
